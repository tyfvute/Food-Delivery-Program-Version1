# 🍕 Food Delivery Program (Version 1)

Программное обеспечение для автоматизации работы современной службы доставки еды.

---

## 🛠️ Функционал приложения
* **Автоматизация заказов:** быстрое оформление, расчет итоговой стоимости и сохранение в базу данных.
* **Управление меню:** просмотр, добавление и редактирование доступных блюд и категорий еды.
* **База данных:** ведение учета клиентов, актуальных адресов и задействованных курьеров.
* **Интерфейс:** интуитивно понятное десктопное приложение для администраторов и операторов.

---

## 🚀 Стек технологий
* **Язык программирования:** C#
* **Платформа** .NET / Windows Forms
* **Среда разработки:** Visual Studio 2022

---

## 📦 Инструкция по запуску
1. Склонируйте этот репозиторий на свой компьютер:
   ```bash
   git clone https://github.com/tyfvute/Food-Delivery-Program-Version1
   ```
2. Откройте файл решения (с расширением `.sln`) в среде Visual Studio.
3. Дождитесь восстановления зависимостей и нажмите **F5** (или кнопку "Пуск") для сборки и запуска.
