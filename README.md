# 🍕 Food Delivery Program (Version 1)

Программное обеспечение для автоматизации работы службы доставки еды. 

## 🚀 Стек технологий
* **Язык программирования:** C#
* **Платформа / UI:** .NET / Windows Forms (или WPF — замените на свой вариант)
* **Среда разработки:** Visual Studio 2022

## 🛠️ Функционал приложения
* Просмотр меню и доступных блюд
* Оформление и расчет стоимости заказа
* Управление базой данных клиентов и ресторанов
* Отслеживание статуса доставки курьерами

## 📦 Инструкция по запуску
1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com
   ```
2. Откройте файл решения `.sln` в Visual Studio.
3. Нажмите `F5` для сборки и запуска проекта.
